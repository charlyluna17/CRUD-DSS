Investigación Aplicada 1: Programación Server-Side y CRUD en PHP

Este proyecto forma parte de la asignatura Desarrollo de Aplicaciones Web con Software Interpretado en el Servidor de la Escuela de Computación. Consiste en una investigación teórica sobre tecnologías del lado del servidor y el desarrollo de una aplicación práctica para la gestión de productos.

🛠️ Descripción del Proyecto

La aplicación es un sistema de gestión de productos desarrollado íntegramente en PHP. Para cumplir con los requerimientos de la guía, la persistencia de datos se maneja mediante una matriz (array) almacenada en la sesión del servidor, prescindiendo totalmente de bases de datos externas.
